Info about the directory structure of the ggtracker uploader project

File/folderDescription
/srcSource folder of ggtracker uploader containing the Java source files and application resources.
/binBinary folder of ggtracker uploader containing the compiled class files and application resources.
The content of this folder is generated by Eclipse automatically, do not put any resources here manually.
/app-folderContains the off-line files and static files required to run ggtracker uploader.
This folder is set as the working directory when running ggtracker uploader from Eclipse.
/release/Application release. Root of release builds and release resource files.
/release-resourcesResource files required to build a release or files of the release.
/ggtracker uploaderThis folder contains the release build.
/build.xmlAnt build script to create a release build.
/directory-info.htmlThis document.
